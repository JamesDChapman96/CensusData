Now make a heat map showing the percentage of couples with each combination of husband race/ethnicity and wife race/ethnicity had a baby in each year. The numbers you calculated in the Table 1 section of Notebook 9 will be the denominators. The numerators will be the number of couples in each group that had a baby in the last year. You will need to start by figuring out which variable indicates that a couple had a baby in the last year and then make a new logical variable for each couple that is TRUE if the couple had a baby in the last year and FALSE otherwise, just as you did in Lab 7 for people who were born in the South. couples %>% mutate(CHILD = FERTYR == 2) %>%

            count(CHILD, wt = PERWT) %>%
            mutate(TOTAL = sum(n)) %>%

Now go through all of the steps you learned in the notebook to assemble couples and match them to their kids so that you end up with the final kids data frame that is ready for graphing. Include comments to indicate what is happening at each step.

#Gather appropriate people
table2 = table2 %>% filter(MARST == 1 & MARRNO == 1 & NCHILD > 0 & (SEXF == "Male" | AGE %in% 20:39))

#Split white/nonwhite parents into two datasets
nonwhite <- table2 %>% filter(RACETH != "NH White")
white <- table2 %>% filter(RACETH == "NH White") %>% select(SERIAL, PERNUM)

#Combine datasets
couples <- inner_join(nonwhite, white, by = c("SERIAL", "SPLOC" = "PERNUM"))

#Create children dataframe
kids = acs %>% filter(AGE < 18 & SERIAL %in% couples$SERIAL) %>%
  select(SERIAL, KIDRACE = RACE, KIDRACED = RACED, KIDH = HISPAN) %>%
  left_join(couples, by = "SERIAL")

#Classify children appropriately
kids <- kids %>% mutate(KID = ifelse(KIDRACE == 1 & KIDH == 0, 1,
                              ifelse(KIDH > 0 & KIDRACE != 1 & RACETH == "Hispanic", 2,
                              ifelse(KIDH == 0 & KIDRACE == 2 & RACETH == "NH Black", 2,
                              ifelse(KIDH == 0 & KIDRACE == 3 & RACETH == "NH Native", 2,
                              ifelse(KIDH == 0 & KIDRACE %in% 4:6 & RACETH == "NH Asian/Pacific", 2,
                              ifelse(KIDRACED %in% 801:823 | (KIDRACE == 1 & KIDH > 0), 3, 4))))))) %>% 
  filter(KID < 4) %>%
  mutate(KID = factor(KID, labels = c("White", "Nonwhite", "Mixed")))
